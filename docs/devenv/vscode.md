# Setup VS Code

When working in GitHub Codespace, VS Code automatically installs recommended extensions.

## Recommended extensions

Otherwise, install recommended extensions:

Code formatting
- [EditorConfig](https://marketplace.visualstudio.com/items?itemName=EditorConfig.EditorConfig)
- [Oxc](https://marketplace.visualstudio.com/items?itemName=oxc.oxc-vscode)

Java support
- [Language Support for Java™ by Red Hat](https://marketplace.visualstudio.com/items?itemName=redhat.java)
- [Debugger for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-debug)
- [Test Runner for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-test)
- [Gradle for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-gradle)

Database
- [Database Client](https://marketplace.visualstudio.com/items?itemName=cweijan.vscode-database-client2)

E2E tests support
- [Cucumber](https://marketplace.visualstudio.com/items?itemName=cucumberopen.cucumber-official)
- [Playwright Test for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-playwright.playwright)
